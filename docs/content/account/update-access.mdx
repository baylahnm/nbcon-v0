## Update Access

Control who can access and modify your projects.

### Access Levels

- **Owner**: Full control
- **Admin**: Can manage users
- **Editor**: Can edit projects
- **Viewer**: Read-only access

### Inviting Users

1. Go to Project Settings
2. Click "Invite Members"
3. Enter email addresses
4. Select access level

### Permissions

Configure granular permissions:

- View projects
- Edit code
- Manage settings
- Delete projects

### Team Management

Organize users into teams for easier management.
