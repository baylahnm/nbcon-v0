## Supported Languages

NBCON PRO supports a wide range of programming languages.

### Fully Supported

- TypeScript / JavaScript
- Python
- Java
- C# / .NET
- Go
- Rust

### Experimental Support

- Kotlin
- Swift
- PHP
- Ruby

### Language Features

- Syntax highlighting
- Auto-completion
- Error detection
- Code formatting

### Adding Language Support

Request new language support or contribute via extensions.
