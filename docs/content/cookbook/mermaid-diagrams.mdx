## Mermaid Diagrams

Create diagrams using Mermaid syntax.

### Supported Diagrams

- Flowcharts
- Sequence diagrams
- Class diagrams
- State diagrams

### Example

\`\`\`mermaid
graph TD
    A[Start] --> B{Decision}
    B -->|Yes| C[Action]
    B -->|No| D[End]
\`\`\`

### Usage

Use Mermaid in:

- Documentation
- Comments
- Markdown files

### Rendering

Diagrams are automatically rendered in the editor and documentation.
