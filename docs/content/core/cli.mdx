## CLI Tool

The NBCON PRO CLI provides powerful command-line tools.

### Installation

```bash
npm install -g @nbcon/cli
```

### Basic Commands

```bash
# Authenticate
nbcon auth login

# Create a project
nbcon project create my-project

# Deploy to cloud
nbcon deploy
```

### Configuration

```bash
# Set API key
nbcon config set apiKey YOUR_KEY

# View configuration
nbcon config get
```
