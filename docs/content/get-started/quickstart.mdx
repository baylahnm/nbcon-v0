## Quickstart Guide

This guide will help you get started with NBCON PRO quickly.

### Installation

```bash
npm install @nbcon/pro
```

### Basic Usage

```typescript
import { NbconClient } from '@nbcon/pro';

const client = new NbconClient({
  apiKey: process.env.NBCON_API_KEY,
});
```
