## Deeplinks

Create shareable links to specific parts of your projects.

### Creating Deeplinks

Right-click on any element and select "Copy Link":

- Files
- Functions
- Code blocks
- Documentation sections

### Using Deeplinks

Paste a deeplink URL to navigate directly to that location:

```
https://nbcon.pro/docs/my-project/src/components/Button.tsx#L42
```

### Share Links

Share deeplinks with your team for:

- Code reviews
- Documentation
- Bug reports
- Feature discussions
