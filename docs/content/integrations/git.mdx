## Git Integration

NBCON PRO integrates seamlessly with Git version control.

### Features

- **Commit Analysis**: AI analyzes commits automatically
- **Branch Management**: Intelligent branch suggestions
- **Merge Assistance**: AI helps resolve merge conflicts

### Git Commands

```bash
# Initialize NBCON PRO in repo
nbcon init

# Analyze changes
nbcon git analyze

# Create commit message
nbcon git commit -m "Your message"
```

### Configuration

Configure Git integration in `.nbcon/config.yml`:

```yaml
git:
  autoCommit: true
  branchStrategy: 'smart'
```
