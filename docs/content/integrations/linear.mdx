## Linear Integration

Sync your Linear issues with NBCON PRO projects.

### Setup

1. Connect your Linear workspace
2. Map Linear projects to NBCON PRO projects
3. Configure sync settings

### Features

- **Issue Sync**: Automatic sync of Linear issues
- **AI Analysis**: AI analyzes issues and suggests fixes
- **Auto-assignment**: Automatically assign issues based on AI analysis

### Configuration

```yaml
linear:
  workspace: your-workspace
  projects:
    - linear-project: nbcon-project
```
