# Pages Documentation Structure

This directory contains implementation plans and documentation for all pages in the nbcon.ai website, organized by page category.

## Directory Structure

```
pages/
├── Resources/          # Resource-related pages (Blog, Forum, Community, etc.)
│   ├── README.md
│   ├── BLOG_PAGE_PLAN.md
│   ├── FORUM_PAGE_PLAN.md
│   ├── COMMUNITY_PAGE_PLAN.md
│   ├── CAREERS_PAGE_PLAN.md
│   ├── CHANGELOG_PAGE_PLAN.md
│   ├── FAQ_PAGE_PLAN.md
│   └── DOCS_ENHANCEMENT_PLAN.md
├── Home/              # Home/Landing page documentation
│   └── README.md
├── Templates/         # Template pages documentation
│   └── README.md
├── Features/          # Features page documentation
│   └── README.md
├── Enterprise/        # Enterprise page documentation
│   └── README.md
├── Pricing/           # Pricing page documentation
│   └── README.md
├── iOS/               # iOS page documentation
│   └── README.md
└── Students/          # Students page documentation
    └── README.md
```

## Categories

### Resources (`/Resources`)
Contains plans for resource and community-related pages:
- Blog
- Forum
- Community
- Careers
- Changelog
- FAQ
- Docs Enhancements

See [Resources/README.md](./Resources/README.md) for detailed plans.

### Home (`/Home`)
Documentation for the main landing/home page.

### Templates (`/Templates`)
Documentation for template-related pages.

### Features (`/Features`)
Documentation for the features page.

### Enterprise (`/Enterprise`)
Documentation for the enterprise page.

### Pricing (`/Pricing`)
Documentation for the pricing page.

### iOS (`/iOS`)
Documentation for the iOS app page.

### Students (`/Students`)
Documentation for the students page.
