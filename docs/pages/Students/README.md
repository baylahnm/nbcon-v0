# Students Page Documentation

## Overview
Documentation and plans for the Students page (`/students`) of nbcon.ai, specifically for KSA (Kingdom of Saudi Arabia) students.

## Current Status
The Students page is fully implemented with:
- SimpleHeroSection component
- Active Universities section (BentoGrid)
- Benefits section
- Eligibility & Verification section
- FAQ section
- Waitlist form

## Structure
- Page specifications
- Student verification process
- University list documentation
- Form handling documentation
- Enhancement plans

## Key Sections
1. **Hero Section** - Uses `SimpleHeroSection` with student-specific CTAs
2. **Active Universities** - BentoGrid showing supported KSA universities
3. **Benefits** - What students get with free Pro access
4. **Eligibility & Verification** - Requirements and verification process
5. **FAQ** - Frequently asked questions about student program
6. **Waitlist** - Form for schools not yet supported

## Components Used
- `SimpleHeroSection` - Hero section
- `BentoGrid` / `BentoGridItem` - University and benefit cards
- `Accordion` - FAQ section
- Form components for waitlist submission

## Typography
All sections use unified typography classes:
- `subsection-heading` for section titles
- `body-large` for descriptions

## Student Program Details
- 1 year of Pro access free
- Requires .edu.sa email or student ID verification
- Available for undergraduate, graduate, and research students
- Non-transferable, one redemption per person
